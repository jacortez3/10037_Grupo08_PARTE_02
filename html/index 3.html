Animación DOM


Las animaciones DOM se refieren a la capacidad de aplicar efectos de animación a elementos del Document Object Model (DOM) utilizando JavaScript y CSS. Estas animaciones permiten cambiar las propiedades y características visuales de los elementos de una página web de manera dinámica y fluida.

var box = document.querySelector('.animated-box');
box.addEventListener('click', function() {
box.style.animationPlayState = 'paused';
});

Eventos DOM

EVENTO ONMOUSE

Los eventos DOM son acciones o sucesos que ocurren en un documento HTML y que pueden ser detectados y manejados mediante JavaScript. Estos eventos pueden ser desencadenados por la interacción del usuario, como hacer clic en un elemento, mover el ratón sobre un elemento, presionar una tecla en el teclado, etc. Los eventos DOM permiten la interactividad y la capacidad de respuesta en las páginas web, ya que se pueden asociar funciones o scripts a un evento específico y ejecutar acciones en respuesta a dicho evento.

function mOver(obj) {
obj.innerHTML = "Gracias"
obj.style.backgroundColor = "#fffff";
}
function mOut(obj) {
obj.innerHTML = "Poner el mouse encima"
obj.style.backgroundColor = "#1ec5e5"
}

Eventlistener DOM

Haz clic

Un EventListener DOM es una función que se adjunta a un elemento HTML para escuchar un determinado evento. Se utiliza para registrar o suscribir una función que se ejecutará cuando ocurra el evento deseado en el elemento seleccionado. Al utilizar un EventListener, podemos detectar y responder a eventos como clics de ratón, cambios de teclado, cambios en el estado de un formulario, desplazamientos de scroll, entre otros. El EventListener se puede agregar a un elemento utilizando el método addEventListener() y proporcionando el tipo de evento a escuchar y la función que se ejecutará cuando ocurra el evento.

El DOM (Modelo de Objetos del Documento) es una representación estructurada y jerárquica de un documento HTML/XML que proporciona una interfaz para acceder y manipular los elementos del documento. En el DOM, cada elemento, atributo, texto o cualquier otro componente del documento se considera un "nodo". Los nodos pueden tener relaciones padre/hijo, hermano/hermano o ancestro/descendiente. A través del DOM, podemos crear, modificar o eliminar nodos, así como acceder a sus propiedades y atributos. Los nodos DOM permiten una manipulación dinámica y programática de la estructura del documento HTML, lo que brinda flexibilidad y posibilidades de interacción con JavaScript.

function myFunction() {
document.getElementById("p1").remove();
}

Coleccion DOM

Una colección DOM HTML es una colección de elementos en un documento HTML a los que se puede acceder y manipular usando JavaScript. Las colecciones se pueden crear utilizando el método document.querySelectorAll(), que toma un selector CSS como argumento.

function Function() {
const myCollection = document.getElementsByTagName("p");
for (let i = 0; i < myCollection.length; i++) {
myCollection[i].style.color = "red";
} }

Lista de Nodos DOM


Una lista de nodos y una colección de HTML son prácticamente lo mismo. Ambos son colecciones tipo matriz (listas) de nodos (elementos) extraídos de un documento. Se puede acceder a los nodos por números de índice. El índice comienza en 0. Ambos tienen una propiedad de longitud que devuelve el número de elementos de la lista (colección). Una HTMLCollection es una colección de elementos de documento.Una lista de nodos es una colección de nodos de documentos (nodos de elementos, nodos de atributos y nodos de texto). Se puede acceder a los elementos de HTMLCollection por su nombre, identificación o número de índice.
Solo se puede acceder a los elementos de NodeList por su número de índice. Una HTMLCollection es siempre una colección en vivo. Ejemplo: si agrega un elemento li a una lista en el DOM, la lista en HTMLCollection también cambiará.
Una NodeList suele ser una colección estática. Ejemplo: si agrega un elemento li a una lista en el DOM, la lista en NodeList no cambiará.
Los métodos getElementsByClassName() y getElementsByTagName() devuelven una HTMLCollection activa.
El querySelectorAll() método devuelve una lista de nodos estática. La childNodes propiedad devuelve una lista de nodos activa.


const myNodelist = document.querySelectorAll("p");
document.getElementById("demo").innerHTML = "Esta Documento tiene " + myNodelist.length + " parrafos.";

Navegación DOM


Navegación DOM HTML es una técnica para navegar a través del Modelo de objetos de documento (DOM) de un documento HTML utilizando JavaScript. El DOM es una estructura en forma de árbol que representa los elementos de un documento HTML. Navegación DOM HTML le permite acceder y manipular elementos en el árbol DOM.
Hay varias formas de navegar por el árbol DOM usando JavaScript. Una forma común es usar el método document.querySelector() para seleccionar un elemento por su selector CSS. Una vez que haya seleccionado un elemento, puede acceder a sus propiedades y métodos para manipularlo.


document.getElementById("id02").innerHTML = document.getElementById("id01").innerHTML;
